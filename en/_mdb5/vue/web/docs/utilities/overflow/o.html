Use these shorthand utilities for quickly configuring how content overflows an element.

Basic example

Barebones overflow functionality is provided for two values by default, and they are not responsive.

This is an example of using .overflow-auto on an element with set width and height dimensions. By design, this content will vertically scroll.
This is an example of using .overflow-hidden on an element with set width and height dimensions.
This is an example of using .overflow-visible on an element with set width and height dimensions. The content renders outside the element's box. This is also default behavior.
This is an example of using .overflow-scroll on an element with set width and height dimensions.

Using Sass variables, you may customize the overflow utilities by changing the $overflows variable in _variables.scss.
