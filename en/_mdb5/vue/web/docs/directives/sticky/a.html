- API

Import

import { mdbSticky } from "mdb-vue-ui-kit";

Properties

Property Type Default Description
animationSticky String '' Set sticky animation
animationUnsticky String '' Set unsticky animation
boundary Boolean false set to true to stop sticky on the end of the parent
delay Number 0 Set the number of pixels beyond which the item will be pinned
direction String 'down' Set the scrolling direction for which the element is to be stikcky
media Number 0 Set the minimum width in pixels for which the sticky should work
offset Number 0 Set the distance from the top edge of the element for which the element is sticky
position String 'top' Set the edge of the screen the item should stick to
