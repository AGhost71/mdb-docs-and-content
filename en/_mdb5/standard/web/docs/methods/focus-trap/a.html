Focus trap - API

Usage

Via JavaScript

const newTrapEl = document.querySelector('.focus-trap'); const newTrapInstance = new mdb.FocusTrap(newTrapEl);

Options

Name Type Default Description
event String 'blur' Event on the last element, which is supposed to fire focus on the first element
condition Function () => true Function which takes an event as an argument. It returns Booleande determining whether to focus the first element (e.g. if the keyCode is suitable)
selector String "button, a, input, select, textarea, [tabindex]:not([tabindex="-1"])" Selector for focusable elements
onlyVisible Boolean true If set to true the focus trap won't trap elements with display: none and visibility: hidden

Methods

Name Description Example
trap Trap focus on the element FocusTrap.dispose()
dispose Manually dispose of component FocusTrap.dispose()
update Recalculates elements and event listeners - doesnt add general event listener on tab FocusTrap.update()
