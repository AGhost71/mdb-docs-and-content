This guide will take you step by step on how to create your project and deploy using MDB CLI.

Express

MDB GO allows you to host any npm project / library

Initialize

Let's initialize npm project first:

$ mdb blank init

Specify project name (i.e. my-express-app) and choose an npm and . You can also add other meta descriptions or accept default values.

Enter project location

$ cd my-express-app

Install express

$ npm install express --save

Open project in editor (ie. VS Code), create index.js and create your app. You can use sample code:

const express = require('express') const app = express() const port = 3000 app.get('/', (req, res) => { res.send('Hello World!') }) app.listen(port, () => { console.log(`Example app listening at http://localhost:${port}`) })

Publish

Once you finish building your project, use the following command do deploy it:

$ mdb backend publish -p node12

Done! Your project will be ready. If you want to check project console use mdb logs command.
