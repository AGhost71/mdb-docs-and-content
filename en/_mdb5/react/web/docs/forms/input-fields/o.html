Block-level or inline-level form text can be created using .form-text.

Associating form text with form controls
Form text should be explicitly associated with the form control it relates to using the aria-describedby attribute. This will ensure that assistive technologiesâ€”such as screen readersâ€”will announce this form text when the user focuses or enters the control.
