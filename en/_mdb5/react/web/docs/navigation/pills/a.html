- API

Import

import { MDBTabs, MDBTabsItem, MDBTabsLink, MDBTabsContent, MDBTabsPane } from 'mdb-react-ui-kit';

Properties

MDBTabs

Name Type Default Description Example
className String '' Add custom class to MDBTabs <MDBTabs className="class">...</MDBTabs>
justify Boolean '' Sets equal width of the MDBTabs items <MDBTabs justify>...</MDBTabs>
fill Boolean '' Fills the available space in MDBTabs items <MDBTabs fill>...</MDBTabs>
pills Boolean '' Necessary to change the MDBTabs into pills <MDBTabs pills>...</MDBTabs>

MDBTabsContent

Name Type Default Description Example
tag String 'div' Defines tag of the MDBTabsContent element <MDBTabsContent tag="section" />
className String '' Add custom class to MDBTabsContent <MDBCardHeader className="class" />

MDBTabsItem

Name Type Default Description Example
className String '' Add custom class to MDBTabsItem <MDBTabsItem className="class" />

MDBTabsLink

Name Type Default Description Example
className String '' Add custom class to MDBTabsLink <MDBTabsLink className="class" />
active Boolean '' Defines an active state of the MDBTabsLink <MDBTabsLink active />
color String '' Allows to change the color of the MDBTabsLink <MDBTabsLink color='danger' />

MDBTabsPane

Name Type Default Description Example
tag String 'div' Defines tag of the MDBTabsPane element <MDBTabsPane tag="section" />
className String '' Add custom class to MDBTabsPane <MDBTabsPane className="class" />
show Boolean '' Defines MDBTabsPanes content is shown <MDBTabsPane show />
