- API

Import

import { MDBTypography } from 'mdb-react-ui-kit';

Properties

MDBTypography

Name Type Default Description Example
tag String 'span' Defines tag of the MDBTypography element <MDBTypography tag="section" />
className String '' Add custom class to MDBTypography <MDBTypography className="class" />
variant String '' Defines a variant for the MDBTypography <MDBTypography variant="h1" />
colorText String '' Sets the color of the text inside the MDBTypography <MDBTypography colorText="primary" />
blockquote Boolean '' Creates a blockquote <MDBTypography blockquote />
note Boolean '' Creates a note <MDBTypography note />
noteColor String '' Sets color of the note <MDBTypography note noteColor="primary" />
listUnStyled Boolean '' Creates an unstyled list from the MDBTypography <MDBTypography listUnStyled />
listInLine Boolean '' Creates an inline list from the MDBTypography <MDBTypography listInLine />
