- API

Import

import { MDBSpinner } from 'mdb-react-ui-kit';

Properties

MDBSpinner

Name Type Default Description Example
tag String 'div' Defines tag of the MDBSpinner element <MDBSpinner tag="section" />
color String '' Defines the color of the MDBSpinner element <MDBSpinner color="secondary" />
className String '' Add custom class to MDBSpinner <MDBSpinner className="class" />
size String '' Sets the size of the spinner <MDBSpinner size="sm" />
grow Boolean '' Makes a growing spinner <MDBSpinner grow />
