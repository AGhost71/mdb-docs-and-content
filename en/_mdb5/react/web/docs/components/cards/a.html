Properties

MDBCard

Name Type Default Description Example
tag String 'div' Defines tag of the MDBCard element <MDBCard tag="section" />
className String '' Add custom class to MDBCard <MDBCard className="class" />
border String '' Sets a border for a card <MDBCard border="secondary" />
shadow String '' Sets a shadow for a card <MDBCard shadow="1" />
background String '' Sets a background color for a card <MDBCard background="secondary" />
alignment String '' Defines a text alignment inside the card <MDBCard alignment="end" />
