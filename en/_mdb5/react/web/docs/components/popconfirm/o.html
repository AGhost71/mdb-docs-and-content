Note: Read the API tab to find all available options and advanced customization

Display mode

You can choose between modal and inline to modify display mode.

Modal mode is rendered at the center of the screen and is static, you can't change its position but all other attributes can be applied

import React, { useState } from 'react'; import { MDBPopcofirm, MDBPopconfirmMessage, MDBPopconfirmButtons, MDBBtn } from 'mdb-react-ui-kit'; export default function App() { const [inlineState, setInlineState] = useState(false); const [modalState, setModalState] = useState(false); return ( <> Are you sure? setModalState(false)}> Cancel setModalState(false)}> Ok Are you sure? setInlineState(false)}> Cancel setInlineState(false)}> Ok ); }
