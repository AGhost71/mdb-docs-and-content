排列元素

使用边缘定位实用程序轻松排列元素。格式是 {property}-{position} 。

其中 property 是以下之一：

  • top -用于垂直 top 位置
  • left -用于水平 left 的位置
  • 底部-用于垂直的底部位置
  • right -用于水平 right 的位置

其中 position 是以下之一：

  • 0 -用于 0 边缘位置
  • 50 -用于 50％边缘位置
  • 100 -用于 100％边缘位置

您可以通过将条目添加到 $ position-values Sass中来添加更多位置值 地图变量。

 .position-relative-example { height: 200px; width: 100%; background-color: #f5f5f5; } .position-relative-example div { width: 2em; height: 2em; background-color: #343a40; border-radius: .25rem; }

中心元素

此外，您还可以使用transform Utility类将元素居中 .translate-middle 。

此类应用转换 translateX（-50％）和 translateY（-50％）到元素，结合边缘定位 实用程序，可让您将元素绝对居中。

 .position-relative-example { height: 200px; width: 100%; background-color: #f5f5f5; } .position-relative-example div { width: 2em; height: 2em; background-color: #343a40; border-radius: .25rem; }
