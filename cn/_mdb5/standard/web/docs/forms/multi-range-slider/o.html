多范围滑块-Bootstrap 5和Material Design 2.0组件

MDBootstrap滑块是一个交互式组件，使用户可以快速滑动 可能的值分布在所需范围内。。

注意：阅读 API 标签以找到所有可用选项，然后 高级定制

基本例子

如果您将class multi-range-slider添加到您的类别中，则多范围滑块滑块将自动初始化 元件。多范围滑块以最大100和最小0值开头。

 p { color: blue; } var myModal = new bootstrap.Modal(document.getElementById('myModal'), options)

一个范围

您可以使用numberOfRanges选项或data-mdb-attr为滑块设置一个范围。

 const oneRange = document.querySelector('.multi-ranges-one'); const oneRangeValue = document.querySelector('#multi-ranges-value'); const oneRangeinit = new mdb.MultiRangeSlider(oneRange, { range: { max: 300, min: 0, }, });

起始值

您可以使用选项startValues将起始值更改为范围。

 const startValue = document.querySelector('.multi-ranges-start-values'); const startValueValues = document.querySelector('#multi-ranges-start-values-show'); const startValueInit = new mdb.MultiRangeSlider(startValue, { startValues: [40, 80], });

工具提示

您可以使用选项工具提示将工具提示设置为范围。

 const tooltips = document.querySelector('#multi-ranges-tooltips'); const initTooltips = new mdb.MultiRangeSlider(tooltips, { tooltips: true, startValues: [0, 100] });

步骤示例

您可以使用选项step将步长设置为范围。

 const step = document.querySelector('#multi-ranges-step'); const initStep= new mdb.MultiRangeSlider(step, { step: true });
