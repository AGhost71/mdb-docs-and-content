了解如何使用一些对齐，排序和偏移选项来修改列 多亏了我们的flexbox网格系统。另外，请参阅如何使用列类来管理宽度 非网格元素。

Note: 务必 阅读“网格”页面 首先，深入研究如何修改和自定义网格列。

这个怎么运作

  • 列建立在网格上’s flexbox体系结构。 Flexbox意味着我们可以选择更改单个列和修改组 行级别的列。您可以选择列的增长，收缩或其他更改方式。

  • 在构建网格布局时，所有内容都在列中。 Bootstrap的层次结构p’s 网格从 容器 行到列到您的内容。在极少数情况下，您可以结合使用内容和列， 但请注意可能会产生意想不到的后果。

  • Bootstrap包含用于创建快速，响应式布局的预定义类。 用 六个断点 在每个网格层有十二列，我们已经为您构建了数十个类 创建所需的布局。如果愿意，可以通过Sass禁用它。

抵消列

您可以通过两种方式抵消网格列：我们的响应式 .offset-网格类和我们的 保证金实用程序。 网格类的大小可匹配列，而边距对于快速布局更有用 偏移的宽度是可变的。
