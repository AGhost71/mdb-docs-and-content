为了更快地进行适合移动设备的响应式开发，Bootstrap包含数十种实用程序 用于显示，隐藏，对齐和间隔内容的类。

变更显示

使用我们的 显示实用程序 用于响应地切换 display 属性的通用值。与我们的混合 网格系统，内容或组件，以在特定视口中显示或隐藏它们。

Flexbox选项

MDBootstrap是使用flexbox构建的，但并非每个元素的 display 已更改为 display：flex ，因为这会增加很多 不必要的覆盖，并意外更改浏览器的主要行为。大多数 我们的组件 内置启用了flexbox。

如果您需要向元素添加 display：flex ，请使用 .d-flex 或响应式变体之一（例如， .d-sm-flex ）。你会 需要此类或 display 值，以允许使用我们的额外内容 flexbox 用于调整大小，对齐，间距等的实用程序。

保证金和填充

使用 margin 和 padding 间隔实用程序 控制元素和组件的间距和大小。Bootstrap包含六个级别 间距实用程序的缩放比例，基于默认的 1rem 值 $ spacer 变量。选择所有视口的值（例如， .me-3 margin-right：1rem ），或选择自适应变体以定位特定视口 （例如， margin-right：1rem 的 .me-md-3 md 断点）。
