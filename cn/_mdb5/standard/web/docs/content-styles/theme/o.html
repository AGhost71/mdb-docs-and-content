MDB Pro component

主题系统使您可以自定义所有MDB组件的外观。

创建一个新主题

创建主题需要重新编译scss样式，为此，我们建议使用 MDB Webpack Starter

创建新主题要求您定义自己的原色和副色 应用。我们准备了函数和mixin，它们将帮助您创建可立即使用的函数 使用这些颜色的主题。

转到使用 MDB Webpack Starter 创建的项目，打开 位于 src / scss / 文件夹中的 index.scss 文件，并替换代码 有以下代码：

@import '~mdb-ui-kit/src/scss/mdb.pro.scss'; $my-theme-primary: #9c27b0; // theme primary color, change this value to customize theme $my-theme-secondary: #69f0ae; // theme secondary color, change this value to customize theme $my-theme: mdb-light-theme($my-theme-primary, $my-theme-secondary); // create the new theme using primary and secondary colors // include theme styles @include mdb-theme($my-theme);

浅色主题

可以使用 mdb-light-theme 函数创建一个浅色主题。你刚才 需要定义 primary 和 secondary 颜色以及所有其他参数 将自动调整。

@import '~mdb-ui-kit/src/scss/mdb.pro.scss'; $my-theme-primary: #9c27b0; // theme primary color, change this value to customize theme $my-theme-secondary: #69f0ae; // theme secondary color, change this value to customize theme $my-light-theme: mdb-light-theme($my-theme-primary, $my-theme-secondary); // create the new light theme using primary and secondary colors // include theme styles @include mdb-theme($my-light-theme);

黑暗主题

可以使用 mdb-dark-theme 函数创建深色主题。您只需 定义 primary 和 secondary 颜色，所有其他参数将是 自动调整。

@import '~mdb-ui-kit/src/scss/mdb.pro.scss'; $my-theme-primary: #9c27b0; // theme primary color, change this value to customize theme $my-theme-secondary: #69f0ae; // theme secondary color, change this value to customize theme $my-dark-theme: mdb-dark-theme($my-theme-primary, $my-theme-secondary); // create the new dark theme using primary and secondary colors // include theme styles @include mdb-theme($my-dark-theme);
