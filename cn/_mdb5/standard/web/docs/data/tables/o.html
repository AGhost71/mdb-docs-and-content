嵌套如何工作

为了防止 any 样式泄漏到嵌套表中，我们使用子组合器 (>) CSS中的选择器。由于我们需要定位所有 td 和 thead ， tbody 和 tfoot 中的第 th 个， 没有它，选择器看起来会很长。因此，我们使用看起来很奇怪的外观 .table > :not(caption) > * > * 选择器定位所有 td 和 .table 的 th ，但没有任何潜在的嵌套表。

请注意，如果您添加<tr>s 作为桌子的直子 <tr> 将被包裹在 <tbody> 默认情况下，因此 使我们的选择器按预期工作。
