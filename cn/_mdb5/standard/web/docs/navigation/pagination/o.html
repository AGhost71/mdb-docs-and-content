用于显示分页以指示一系列相关内容的文档和示例 存在于多个页面中。

基本例子

分页是使用列表HTML元素构建的，因此屏幕阅读器可以宣布 可用链接。使用包装纸 <nav> 将其标识为一个元素 屏幕阅读器和其他辅助技术的导航部分。

另外，由于页面可能包含多个导航部分，因此建议您 提供描述性的 aria-label <nav> 反映其 目的。例如，如果分页组件用于在一组搜索之间导航 结果，适当的标签可以是 aria-label =“搜索结果页” 。

禁用和活动状态

分页链接可针对不同情况进行定制。将 .disabled 用于 链接显示为不可点击，并且 .active 表示当前页面。

虽然 .disabled 类使用 pointer-events：none 来 try 禁用CSS属性的＆lt; a＆gt; s的链接功能 尚未标准化，也无法说明键盘导航。因此，您应该 始终在禁用的链接上添加 tabindex =“-1” 并使用自定义JavaScript完全 禁用其功能。
