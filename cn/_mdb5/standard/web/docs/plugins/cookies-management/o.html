MDB Pro component

该组件用于在浏览器中保存我们希望下次访问的数据 用户访问。例如，用户上次访问我们的页面的次数或时间。

注意：阅读 API 标签以找到所有可用选项，然后 高级定制

基本例子

Cookies管理组件提供添加，删除和获取Cookies的方法

设定Cookie

使用方法 Cookie.set（）添加一个cookie。您可以使用 下面的例子。该按钮将调用 Cookie.set（“首次访问”，新的Date（），{过期：1}）。要再次使用此示例， 按重置按钮并刷新页面。

Cookie.set('first-visit', new Date(), {expires: 1})

获取cookie

如果您已在用户的Cookie中保存了任何数据，则可以使用 Cookie.get 方法。

Cookie.get('cookie-name');

删除cookie

当不再需要保存在cookie中的数据并且想要删除它们时，请使用 Cookie.remove 方法。刷新页面后，cookie将被删除

Cookie.remove('cookie-name');
