色彩

MDB包含几种预定义的按钮样式，每种样式都有其自己的语义目的，并且 很少有其他东西可以控制。

向辅助技术体现含义：
使用颜色添加含义仅提供视觉指示，不会传达给 辅助技术的用户-例如屏幕阅读器。确保信息表示为 颜色可以从内容本身（例如可见文字）中显而易见，也可以包含在内 通过其他方式，例如用 .visually-hidden 类。
